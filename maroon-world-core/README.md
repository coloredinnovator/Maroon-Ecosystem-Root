# Maroon World Core

> **Codex Reference:** §4.1 — Gaming Platform

Open-source gaming engine built on Godot with C++ GDExtension bindings. Features an in-game Asset Marketplace using Maroon Currency.

## Features
- Godot GDExtension C++ bindings
- In-game Asset Marketplace (Maroon Currency)
- Player session management
- Gaming telemetry → Palantir Lake

## Tech Stack
- **Language:** C++17
- **Engine:** Godot (GDExtension)
- **Build:** CMake
