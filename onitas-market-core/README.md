# Onita's Market Core

> **Codex Reference:** §4.2 — First-Party Grocery Storefront

React/Next.js frontend consuming Maroon Market APIs. Features EBT-eligible product catalog with nutritional data.

## Features
- Product catalog with EBT eligibility badges
- Split-Tender cart (EBT vs Fiat separation)
- Sovereign Checkout → Maroon Market Core API
- Palantir telemetry on all user interactions

## Tech Stack
- **Language:** TypeScript/JSX
- **Framework:** React + Next.js
- **Port:** 3000
