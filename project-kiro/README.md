# Project Kiro

This directory represents **Project Kiro** within the Maroon Ecosystem.

## AWS Kiro IDE & CLI

Kiro is the AWS Agentic IDE built on Code OSS. 

We have successfully installed the Kiro CLI onto this environment. 
You can run the Kiro CLI directly in your terminal using the `kiro-cli` command.

### Installation Details
- **CLI Executable Path:** `C:\Users\maroon\AppData\Local\Kiro-Cli\kiro-cli.exe`
- **Current Version:** `2.8.1`

### Getting Started
To authenticate and use the Kiro CLI AI assistant in your terminal, run:

```powershell
& "C:\Users\maroon\AppData\Local\Kiro-Cli\kiro-cli.exe" login
& "C:\Users\maroon\AppData\Local\Kiro-Cli\kiro-cli.exe" chat
```
